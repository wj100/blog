code ELIFECYCLE报错

网上搜到的办法都是删除node_modules
但是并没有用
原来是正则表达式的问题
用 new RegExp 的方式去创建即可
将

reg=/(?<=(sem\/p|sem\/m)).*?(?=\.)/g

改为

reg=new RegExp("/(?<=(sem\/p|sem\/m)).*?(?=\\.)/g")

注意.前面加了个\
